## 📜 Descripción  
Es un sistema de escritorio completo diseñado para gestionar asistencias, generar reportes de empleados y administrar configuraciones laborales de manera sencilla y eficiente. ¡Registra entradas/salidas, genera estadísticas y automatiza ausencias! 

La aplicación desarrollada con **Electron + React** que permite:  
- Registrar asistencias manuales o automáticas.
- Gestionar empleados y departamentos.
- Exportar reportes detallados en Excel.  
- Configurar horarios laborales y tolerancias.

*Perfecto para pequeñas/medianas empresas que necesitan un control local sin dependencia de internet.*  

## ✅ Características Principales  
- **Registro** de entradas/salidas.
- **Ausencias automáticas** (según horario configurado).  
- **Filtros avanzados** por fecha, departamento o estado. 
- **Dashboard** intuitivo.
- **Reportes en Excel** con un solo clic.  
- **Interfaz amigable** para el usuario.
